# Guide d'utilisation d'Eventify

## Bienvenue sur Eventify !

Eventify est une application web qui vous permet de découvrir, créer et participer à des événements près de chez vous. Ce guide vous aidera à prendre en main toutes les fonctionnalités de la plateforme.

## Créer un compte

1. Rendez-vous sur la page d'accueil d'Eventify
2. Cliquez sur "Commencer maintenant"
3. Connectez-vous avec votre compte Google en cliquant sur le bouton correspondant
4. Acceptez les autorisations demandées
5. Votre compte est créé ! Vous pouvez maintenant personnaliser votre profil

## Découvrir des événements

### Carte des événements

La page principale d'Eventify affiche une carte interactive avec tous les événements à proximité. Pour l'utiliser :

1. Autorisez la géolocalisation lorsque l'application vous le demande
2. Naviguez sur la carte pour explorer les différentes zones
3. Cliquez sur un marqueur pour voir les détails de l'événement
4. Utilisez le zoom pour ajuster la vue

### Filtres

Pour affiner votre recherche d'événements :

1. Utilisez la barre de recherche "Ville" pour trouver des événements dans une localité spécifique
2. Filtrez par nom d'événement en utilisant le champ "Nom de l'événement"
3. Triez les résultats par date (croissante ou décroissante)
4. Triez par distance (la plus proche ou la plus éloignée)
5. Ajustez le nombre d'événements affichés avec le sélecteur "Affichage"

## Créer un événement

Pour organiser votre propre événement :

1. Cliquez sur votre profil en haut à droite
2. Sélectionnez "Créer un événement"
3. Remplissez le formulaire avec :
   - Le nom et la description de l'événement
   - Une image d'illustration (formats .jpg, .jpeg ou .png)
   - Les dates et heures de début et de fin
   - L'adresse complète et le nom du lieu
4. Cliquez sur "Créer un événement"

Une fois créé, votre événement sera visible sur la carte et dans les résultats de recherche.

## Rejoindre un groupe

Pour rejoindre un groupe existant pour un événement :

1. Accédez à la page de détails de l'événement en cliquant sur "Plus d'informations"
2. Dans la section "Liste des groupes", vous verrez tous les groupes disponibles
3. Cliquez sur "Rejoindre" à côté d'un groupe public
4. Vous êtes maintenant membre du groupe et pouvez accéder au salon de discussion

Note : Les groupes privés nécessitent une invitation du créateur du groupe.

## Créer un groupe

Pour créer votre propre groupe pour un événement :

1. Accédez à la page de détails de l'événement
2. Cliquez sur "Créer un groupe"
3. Remplissez le formulaire avec :
   - Le nom et la description du groupe
   - La visibilité (public ou privé)
   - Les utilisateurs à inviter (optionnel)
4. Cliquez sur "Créer un groupe"

En tant que créateur, vous pourrez gérer les membres et modifier les informations du groupe.

## Utiliser la messagerie

Chaque groupe dispose d'un salon de discussion :

1. Accédez au groupe en cliquant sur "Accéder" depuis la page de l'événement
2. Le salon de discussion s'affiche dans la partie droite de l'écran
3. Tapez votre message dans la zone de texte en bas
4. Appuyez sur Entrée pour envoyer
5. Pour un retour à la ligne sans envoyer, utilisez Maj+Entrée

Les messages sont affichés avec votre pseudo et votre avatar, et restent visibles pour tous les membres du groupe.

## Gérer votre profil

Pour personnaliser votre profil :

1. Cliquez sur "Mon profil" dans la barre de navigation
2. Pour modifier vos informations, cliquez sur "Modifier le profil"
3. Vous pouvez changer :
   - Votre nom
   - Votre pseudo
   - Votre biographie
   - Votre avatar (en cliquant sur l'image actuelle)
4. Cliquez sur "Enregistrer" pour valider les modifications

Votre profil affiche également tous les événements auxquels vous participez ou que vous organisez.

## FAQ

### Comment supprimer mon compte ?
Contactez-nous à support@eventify.com pour demander la suppression de votre compte.

### Comment quitter un groupe ?
Sur la page de détails du groupe, cliquez sur le bouton "Quitter le groupe".

### Comment supprimer un événement que j'ai créé ?
Accédez au tableau de bord de l'événement, puis cliquez sur "Supprimer l'événement".

### Les événements sont-ils modérés ?
Oui, tous les événements sont soumis à nos conditions d'utilisation et peuvent être supprimés s'ils ne les respectent pas.

### Comment signaler un problème ?
Contactez-nous à support@eventify.com en précisant la nature du problème.
